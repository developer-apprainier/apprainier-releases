# Web SDK v1.0.0

## Files

- `apprainier-web-sdk-1.0.0/`
- `apprainier-web-sdk-1.0.0.zip`

## Usage

Use this artifact for websites and browser-based apps.

For local testing, install the extracted SDK from a web app:

```bash
npm install ../apprainier-web-sdk-1.0.0
```

The web app should load the AppRainier config JSON generated from the admin portal and pass it during SDK initialization.

## Notes

- This package is an ES module package.
- The SDK entrypoint is `src/index.js`.
- Type definitions are available at `src/index.d.ts`.
