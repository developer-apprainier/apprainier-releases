# Flutter SDK v2.0.0

## Files

- `flutter_apprainier_plugin-2.0.0`
- `flutter_apprainier_plugin-2.0.0.zip`

## Usage

Use this artifact for Flutter apps.

The plugin bundle includes the refreshed AppRainier Android AAR and iOS XCFramework, so Flutter apps do not need to add the native SDKs separately.

Keep each customer's AppRainier config JSON outside this release artifact and include it in the client app.
