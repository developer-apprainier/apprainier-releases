# Web SDK v2.0.0

## Files

- `apprainier-web-sdk-2.0.0`
- `apprainier-web-sdk-2.0.0.zip`

## Usage

Use this artifact for websites and browser-based apps.

Serve the AppRainier config JSON generated from the admin portal with the website, then initialize the SDK from that config.

## Notes

- The Web SDK is dependency-free and uses scoped `apprainier-` DOM classes.
- Live card carousel text overlays, built-in announcement icons, and numeric/native-style text size handling are included.
- Add the website origin to Appwrite Web/CORS platform settings before testing.
