# React Native SDK v2.0.0

## Files

- `react-native-apprainier-plugin-2.0.0`
- `react-native-apprainier-plugin-2.0.0.zip`

## Usage

Use this artifact for React Native apps.

The plugin bundle includes the refreshed AppRainier Android AAR and iOS XCFramework, so React Native apps do not need to add the native SDKs separately.

Keep each customer's AppRainier config JSON outside this release artifact and include it in the client app.
